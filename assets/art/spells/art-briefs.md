# AniGO Spell Card Art Briefs — For nanobanana

15 spell illustrations across 5 champions. Each spell needs a square card art piece.

## Art Specs

| Detail | Spec |
|--------|------|
| Format | PNG with transparency preferred, or JPG |
| Aspect ratio | 1:1 square |
| Resolution | 512x512px minimum (1024x1024 for retina) |
| Style | Anime/manga digital painting, matching champion art style |
| Lighting | Dramatic, cinematic, matches the champion's color palette |
| Subject | The spell effect itself on/near a Go board — NOT the champion casting it |
| File names | `{spellid}.png` (listed below) |
